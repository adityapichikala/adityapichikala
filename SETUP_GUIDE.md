## üîß Step 2: GitHub Actions Setup

### Enable GitHub Actions

1. Go to your repository settings
2. Navigate to **Actions** ‚Üí **General**
3. Enable **Read and write permissions** for workflows
4. Enable **Allow GitHub Actions to create and approve pull requests**

### Required Secrets

Navigate to **Settings** ‚Üí **Secrets and variables** ‚Üí **Actions** and add:

#### 1. WakaTime API Key
- Sign up at [WakaTime](https://wakatime.com)
- Install WakaTime plugin in your IDE (VS Code, PyCharm, etc.)
- Get your API key from [WakaTime Settings](https://wakatime.com/settings/api-key)
- Add secret: `WAKATIME_API_KEY` = `your_api_key`

#### 2. GitHub Token
- `GITHUB_TOKEN` is automatically provided by GitHub Actions
- No action needed

## üö® Troubleshooting

### Snake Animation Not Showing
1. Check if `output` branch exists
2. Verify workflow ran successfully
3. Check Actions logs for errors
4. Ensure repository is public or you have proper permissions

### WakaTime Not Updating
1. Verify `WAKATIME_API_KEY` secret is set
2. Check if you've coded in the last 7 days
3. Ensure WakaTime plugin is active in your IDE
4. Check Actions logs

### Blog Posts Not Showing
1. Verify RSS feed URLs are correct
2. Check if you have published posts
3. Ensure workflow has write permissions
4. Check Actions logs

### Stats Not Loading
1. Verify username is correct
2. Check if repository is public
3. Try clearing browser cache
4. Verify Vercel APIs are accessible
